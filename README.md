# Sayı Tahmin Oyunu (Number Guessing Game) 🎮

Python ile geliştirilmiş, kullanıcının 1 ile 100 arasında rastgele seçilen bir sayıyı bulmaya çalıştığı basit ve eğlenceli bir konsol kodu. 

Kullanıcının sayıyı bulabilmesi için toplam **5 hakkı** bulunmaktadır ve program her tahminde kullanıcıyı "Daha Yüksek" veya "Daha Düşük" şeklinde yönlendirir.
